# About Horizon Learning

Horizon Learning exists to make complex technology approachable.

Many people want to learn topics like Linux, networking, cybersecurity, programming, and self-hosting, but the learning process can feel scattered and intimidating.

Horizon Learning is designed to provide a smoother path.

## Mission

Make practical technology education easier to understand, easier to follow, and easier to apply.

## Who this is for

Horizon Learning is for people who:

- Are new to technical topics
- Prefer plain-English explanations
- Want step-by-step guidance
- Want to understand what they are doing, not just copy commands
- Want safe hands-on practice
- Want a clear path from beginner to confident learner

## What makes this different

Horizon Learning is not trying to be the biggest library of tutorials on the internet.

The goal is to be organized, practical, and beginner-friendly.

Each learning path is designed to answer:

- What is this?
- Why does it matter?
- How does it work?
- How do I practice it safely?
- What should I learn next?

## Learning approach

Horizon Learning focuses on:

- Fundamentals before advanced topics
- Understanding before memorization
- Practical examples
- Short lessons
- Clear checkpoints
- Safe practice environments
- Responsible technology use

## Current focus

The first learning path is:

- Cybersecurity First Steps

This path introduces cybersecurity, Kali Linux, Linux basics, networking fundamentals, beginner tools, and safe practice labs.

## Long-term direction

Horizon Learning may eventually include more learning paths, downloadable guides, checklists, glossaries, labs, and interactive progress tracking.

For now, the priority is simple:

Build one clear, useful, beginner-friendly learning path at a time.
