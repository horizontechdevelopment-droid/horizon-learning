# Glossary

This glossary explains beginner terms in plain English.

## Cybersecurity

The practice of protecting computers, networks, accounts, and data.

## Kali Linux

A Linux system made for cybersecurity learning, testing, and security work.

## Linux

An operating system, like Windows or macOS, but commonly used for servers, development, and security work.

## Terminal

A text-based way to control the computer by typing commands.

## Command

An instruction typed into the terminal.

## Directory

Another word for folder.

## File path

The location of a file or folder on the computer.

## Root

The highest-level administrator account on Linux.

## Sudo

A command that lets an approved user run something with administrator-level permissions.

## ISO

A disk image file. In this guide, the Kali ISO is used to create a live USB.

## Live USB

A USB drive that can start an operating system without installing it permanently.

## Router

The device that connects your home network to the internet.

## IP address

A number that identifies a device on a network.

## Port

A numbered doorway that a device or program uses for network communication.

## Service

A program running on a device that listens for network connections.

## Scan

A check performed by a tool to learn information about a system or network.

## Vulnerability

A weakness that could cause a security problem.

## Exploit

A method used to take advantage of a vulnerability.
