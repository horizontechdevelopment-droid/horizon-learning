# Learning Path

## Goal

Understand the order you should follow as a beginner.

## Plain-English explanation

Cybersecurity can feel overwhelming because there are many topics, tools, and opinions.

This guide is designed to avoid that problem.

Instead of jumping straight into advanced tools, you will build the basics first.

## Beginner path

Follow this order:

1. Safety and permission
2. What cybersecurity is
3. What Kali Linux is
4. How to download Kali
5. How to create a live USB
6. How to boot into Kali
7. Basic Linux commands
8. Basic networking
9. Kali tool overview
10. Home Wi-Fi security check
11. First Nmap scan
12. Beginner lab roadmap

## Why this order matters

Kali Linux includes many advanced tools.

If you open Kali without understanding Linux or networking basics, it can feel confusing fast.

The goal is to learn what the tools are doing, not just copy commands.

## Recommended pace

Take your time.

A good pace is one or two lessons at a time.

If a lesson feels confusing, repeat it before moving forward.

## Stop and check

You are ready to continue when:

- [ ] You understand the safety rules
- [ ] You know this guide is for authorized learning only
- [ ] You are ready to learn step by step
