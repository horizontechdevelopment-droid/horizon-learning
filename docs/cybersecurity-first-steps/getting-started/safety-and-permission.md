# Safety and Permission

## Goal

Understand where cybersecurity learning is allowed and where it can get you into trouble.

## Plain-English explanation

Cybersecurity tools are powerful. The same tool that helps you check your own network can also be misused against someone else's network.

This guide is only for legal, authorized learning.

## Safe places to practice

You can practice on:

- Your own computer
- Your own Wi-Fi network
- Your own router
- A spare router you set up as a lab
- Intentionally vulnerable training labs
- Practice platforms designed for cybersecurity learning

## Do not test

Do not scan, attack, or probe:

- Someone else's Wi-Fi
- School networks
- Work networks
- Public Wi-Fi
- Random websites
- Friends' devices without clear permission
- Any account or system you do not own

## Checkpoint

Before using any tool, ask:

Do I own this, or do I have clear permission?

If the answer is no, stop.
