# Create a Live USB

## Goal

Create a USB drive that can boot into Kali Linux.

## Plain-English explanation

A live USB lets a computer start Kali from a USB drive instead of starting Windows normally.

This does not mean Kali is installed permanently.

It is a way to test and learn Kali without replacing the main operating system.

## Warning

Creating a live USB will erase the USB drive.

Do not use a USB drive that contains important files.

## What you need

- Kali ISO file
- USB drive
- A USB writing tool such as Rufus or Balena Etcher
- A computer that can boot from USB

## Basic steps

1. Insert the USB drive.
2. Open the USB writing tool.
3. Select the Kali ISO file.
4. Select the USB drive.
5. Confirm that the correct USB drive is selected.
6. Start the write process.
7. Wait until the tool says it is complete.
8. Safely remove the USB drive.

## Common mistake

The most common mistake is selecting the wrong drive.

Before clicking start, double-check the drive letter and size.

## Stop and check

You are ready to continue when:

- [ ] The USB writing tool finished successfully
- [ ] You used the correct USB drive
- [ ] You understand that the USB drive was erased
- [ ] You are ready to boot from USB
