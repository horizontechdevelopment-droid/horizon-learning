# Download Kali

## Goal

Download Kali Linux safely from the official source.

## Plain-English explanation

An ISO file is a disk image.

For this guide, the ISO file is the Kali Linux file that will be used to create the live USB.

## Important safety rule

Only download Kali from the official Kali website.

Do not download Kali from random websites, file mirrors, YouTube descriptions, or unknown links.

## What you need

- A computer with internet access
- A USB drive
- Enough storage space for the download
- Time to let the download finish

## Basic steps

1. Go to the official Kali Linux website.
2. Open the Downloads section.
3. Choose the live USB image.
4. Download the ISO file.
5. Do not rename or modify the file yet.
6. Keep the file somewhere easy to find, such as Downloads.

## Stop and check

You are ready to continue when:

- [ ] You downloaded Kali from the official website
- [ ] You know where the ISO file is saved
- [ ] You are ready to create the live USB
