# First Boot

## Goal

Boot into Kali Linux from the live USB.

## Plain-English explanation

Normally, your computer starts Windows from the internal drive.

For this lesson, you will tell the computer to start from the Kali USB instead.

## Before you start

Save your work in Windows first.

Shut down or restart the computer only when you are ready.

## Basic steps

1. Insert the Kali USB drive.
2. Restart the computer.
3. Open the boot menu.
4. Choose the USB drive.
5. Wait for Kali to load.
6. Choose the live option.
7. Let Kali start.

## Boot menu keys

The boot menu key depends on the computer brand.

Common keys include:

- F12
- F11
- F10
- Esc
- Del

## If it does not boot

Try:

- Restarting and pressing the boot key earlier
- Trying another USB port
- Checking that the USB was created correctly
- Looking up the boot menu key for the computer model

## Stop and check

You are ready to continue when:

- [ ] Kali starts from the USB
- [ ] You can see the Kali desktop
- [ ] You understand this is running from USB
