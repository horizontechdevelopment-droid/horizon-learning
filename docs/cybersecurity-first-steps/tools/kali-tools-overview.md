# Kali Tools Overview

## Goal

Understand the types of tools included with Kali Linux.

## Plain-English explanation

Kali Linux includes many cybersecurity tools.

As a beginner, you do not need to learn all of them at once.

It is better to understand the categories first.

## Tool categories

### Information gathering

These tools help you learn about systems, networks, domains, or services.

Example:

    nmap

### Network analysis

These tools help inspect network traffic.

Example:

    wireshark

### Web application testing

These tools help test websites and web apps in authorized environments.

Examples:

    burp suite
    nikto
    gobuster

### Password auditing

These tools help test password strength in legal lab environments.

Examples:

    john
    hashcat

### Wireless testing

These tools help study Wi-Fi security in authorized lab environments.

Examples:

    aircrack-ng suite

### Forensics

These tools help investigate files, drives, memory, or activity.

Examples:

    autopsy
    binwalk

### Exploitation frameworks

These tools help test known vulnerabilities in legal labs.

Example:

    metasploit

## Beginner tools to start with

Start with:

- Nmap
- Wireshark
- Basic Linux commands
- Browser-based router settings
- Safe training labs

## Tools to avoid at first

Do not start with:

- Password cracking
- Wi-Fi attack tools
- Exploitation frameworks
- Malware analysis tools
- Tools you do not understand

These can come later in controlled lab environments.

## Important rule

A tool is not good or bad by itself.

What matters is how, where, and why it is used.

Only use tools on systems you own or have permission to test.

## Stop and check

You are ready to continue when:

- [ ] You understand Kali tools are grouped by purpose.
- [ ] You know you do not need to learn every tool at once.
- [ ] You know Nmap and Wireshark are good beginner tools.
- [ ] You understand permission still matters.
