# Cybersecurity First Steps

This guide is for someone who is curious about cybersecurity but new to Linux, Kali Linux, networking, and command-line tools.

The goal is not to rush into hacking.

The goal is to learn safely, legally, and step by step.

## How to use this guide

Start at the beginning and move in order.

Each lesson will include:

- Goal
- Plain-English explanation
- Step-by-step instructions
- Checkpoint
- Common mistakes
- What to do next

## First rule

Only test systems, devices, accounts, and networks that you personally own or have clear permission to test.
