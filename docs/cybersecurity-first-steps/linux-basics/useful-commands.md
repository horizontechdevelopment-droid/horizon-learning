# Useful Commands

## Goal

Learn beginner Linux commands that are useful in Kali.

## Plain-English explanation

Commands are tools you type into the terminal.

You do not need to memorize everything right away.

The goal is to understand the most common commands and know how to ask for help.

## Navigation commands

Show current folder:

    pwd

List files:

    ls

Change folder:

    cd folder-name

Go back one folder:

    cd ..

Go to your home folder:

    cd ~

## File commands

Create a folder:

    mkdir practice

Create an empty file:

    touch notes.txt

View a text file:

    cat notes.txt

Copy a file:

    cp notes.txt notes-copy.txt

Rename or move a file:

    mv notes.txt new-name.txt

Delete a file:

    rm new-name.txt

## System information

Show current user:

    whoami

Show computer name:

    hostname

Show network information:

    ip addr

Show running processes:

    ps aux

## Help commands

Show command manual:

    man command-name

Example:

    man ls

Show quick help:

    command-name --help

Example:

    nmap --help

## Safe beginner practice

Create a practice folder:

    mkdir linux-practice

Move into it:

    cd linux-practice

Create a notes file:

    touch notes.txt

List the folder:

    ls -l

Go back:

    cd ..

## Stop and check

You are ready to continue when:

- [ ] You can move between folders.
- [ ] You can create a folder.
- [ ] You can create a file.
- [ ] You know how to ask a command for help.
- [ ] You understand that delete commands should be used carefully.
