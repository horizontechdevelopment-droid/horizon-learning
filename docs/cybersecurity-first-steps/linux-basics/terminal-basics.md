# Terminal Basics

## Goal

Learn what the terminal is and how to use a few basic commands safely.

## Plain-English explanation

The terminal is a text-based way to talk to the computer.

Instead of clicking buttons, you type commands.

Kali Linux uses the terminal a lot because many cybersecurity tools are designed to be controlled with commands.

## New terms

- **Terminal:** A window where you type commands.
- **Command:** An instruction you give the computer.
- **Prompt:** The text that appears before where you type.
- **Output:** The response the computer gives after a command runs.
- **Directory:** Another word for folder.

## Opening the terminal

In Kali, you can usually open the terminal by:

1. Clicking the terminal icon.
2. Or opening the application menu.
3. Searching for Terminal.
4. Clicking it.

## Basic commands

Show your current location:

    pwd

List files and folders:

    ls

Move into a folder:

    cd folder-name

Move back one folder:

    cd ..

Clear the screen:

    clear

Show who you are logged in as:

    whoami

## Important warning

Do not copy and paste random commands from the internet.

Some commands can delete files, change settings, or break things.

If you do not understand what a command does, pause and look it up first.
