# Wi-Fi Basics

## Goal

Understand what Wi-Fi is and what basic Wi-Fi security means.

## Plain-English explanation

Wi-Fi lets devices connect to a network without a cable.

Because Wi-Fi uses radio signals, nearby people may be able to see that the network exists.

That is why Wi-Fi security matters.

## New terms

- **SSID:** The Wi-Fi network name.
- **Password:** The key used to join the Wi-Fi network.
- **Encryption:** A way of protecting information so others cannot easily read it.
- **WPA2:** A common Wi-Fi security standard.
- **WPA3:** A newer Wi-Fi security standard.
- **Guest network:** A separate Wi-Fi network for visitors or less trusted devices.

## Good Wi-Fi security

A safer home Wi-Fi setup usually has:

- WPA2 or WPA3 enabled
- A strong Wi-Fi password
- A strong router admin password
- Router firmware updates installed
- WPS disabled if not needed
- A guest network for visitors or smart devices

## Weak Wi-Fi security

Avoid:

- No password
- WEP security
- Very short passwords
- Passwords based on names, pets, addresses, or birthdays
- Default router admin password
- Old router firmware

## Wi-Fi password vs router admin password

These are different.

The Wi-Fi password lets devices join the network.

The router admin password lets someone change router settings.

Both should be strong.

## Safe beginner mindset

The first Wi-Fi lesson should not be about breaking into Wi-Fi.

The first lesson should be learning how to make your own Wi-Fi safer.

## Stop and check

You are ready to continue when:

- [ ] You know what an SSID is.
- [ ] You know WPA2/WPA3 are better than WEP.
- [ ] You understand that Wi-Fi password and router admin password are different.
- [ ] You understand that your first Wi-Fi goal is security checking, not attacking.
