# Contributing to Horizon Learning

Horizon Learning is a beginner-friendly technology education platform.

The goal is not to create the largest collection of lessons.

The goal is to create clear, practical, trustworthy learning paths that help people understand technology.

## Core Principles

Contributions should follow these principles:

- Explain concepts clearly.
- Assume the learner is intelligent but new.
- Avoid unnecessary jargon.
- Teach the why, not just the steps.
- Include safe hands-on practice when appropriate.
- Keep lessons mobile-friendly.
- Keep cybersecurity content ethical and permission-based.

## Lesson Standard

New lessons should follow:

    internal/templates/lesson-template.md

Lessons should include:

- YAML metadata
- Estimated time
- Difficulty
- Prerequisites
- Learning objectives
- Plain-English explanation
- Try it yourself activity
- Reflection questions
- Common mistakes
- Troubleshooting
- Stop and check
- Lesson summary
- What to do next

## Review Standard

Before a lesson is considered complete, review it using:

    internal/checklists/lesson-review.md

## Internal Standards

Contributors should review:

    internal/standards/curriculum-standard.md
    internal/standards/curriculum-graph-standard.md
    internal/workflows/publishing-workflow.md
    internal/design/design-system.md

## Commit Style

Use small, meaningful commits.

Good examples:

    Add networking basics lesson
    Polish IP addresses and ports lesson
    Add lesson review checklist
    Fix broken lesson link

Avoid vague commits like:

    updates
    stuff
    changes

## Safety Standard

Cybersecurity lessons must only teach safe, authorized practice.

Use this rule:

    Only test systems, networks, accounts, and devices that you own or have clear permission to test.

If a lesson includes commands or tools that could affect a system or network, explain the risk clearly.

## Writing Tone

Horizon Learning should feel:

- Clear
- Calm
- Practical
- Respectful
- Encouraging

It should not feel:

- Condescending
- Sensational
- Overly academic
- Fear-based
- Flashy for no reason

## Publishing Goal

Publish lessons when they are useful and trustworthy, then continue improving them over time.

Horizon Learning is a living curriculum.
