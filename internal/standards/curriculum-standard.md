# Horizon Learning Curriculum Standard

Version: 1.0

---

# Purpose

This document defines the educational standards used throughout Horizon Learning.

These standards exist to ensure every learning path provides a consistent, high-quality learning experience.

---

# Mission

Help people understand technology through clear explanations, practical experience, and responsible learning.

---

# Target Audience

Our primary audience includes:

- Complete beginners
- Career changers
- Hobbyists
- Students
- Professionals learning a new technology

We assume curiosity, not prior experience.

---

# Educational Philosophy

Every lesson should:

- Build confidence.
- Explain concepts before procedures.
- Encourage exploration.
- Reinforce ethical behavior.
- Reduce intimidation.

---

# Reading Level

Target approximately an 8th–10th grade reading level.

Use technical terminology when appropriate, but always explain it.

---

# Lesson Length

Most lessons should require:

10–20 minutes.

Longer topics should be divided into multiple lessons.

---

# Practical Learning

Every lesson should include an activity.

Examples:

- Observe
- Practice
- Experiment
- Document
- Build

Reading alone is not enough.

---

# Reflection

Every lesson should encourage learners to reflect.

Reflection is preferred over quizzes.

---

# Safety

Cybersecurity lessons must reinforce:

Only test systems, devices, accounts, and networks that you own or have explicit permission to test.

---

# Continuous Improvement

Technology changes.

Horizon Learning is a living curriculum.

Lessons should be reviewed regularly for technical accuracy.
